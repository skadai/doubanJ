# doubanJ功能介绍
-----
本项目是学习狗书(flask_web_development)过程的练手，主要参考了书中的博客程序的auth模块
## 电影搜索
1. 搜集了大约10000部豆瓣7.0评分以上的电影
2. 提供检索，收藏，和评论功能
3. 支持导引豆瓣主页和相关下载链接

## 影评博客
1. 针对特定的电影添加影评，影评支持markdown语法
2. 支持对影评进行编辑和添加评论

## 爬虫脚本
1. 脚本基于scrapy框架检测豆瓣上的新增的优秀电影，收录进数据库中
